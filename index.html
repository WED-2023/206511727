    אודות:

    בון ג'ובי היא להקת רוק אמריקאית שהוקמה בשנת 1983 בהובלת ג'ון בון ג'ובי.
    הלהקה ידועה בלהיטים כמו "Livin' on a Prayer" ו-"You Give Love a Bad Name",
    והפכה לאחת מלהקות הרוק הנמכרות ביותר בכל הזמנים.
    ג'ון בון ג'ובי, שהושפע מברוס ספרינגסטין, הוא לא רק מוזיקאי אלא גם פילנתרופ,
    תומך במאבק בחסרי בית ובעוני באמצעות קרן "Jon Bon Jovi Soul Foundation".
    התשוקה שלו לסיפורי חיים ושירות קהילתי מגדירה גם את המוזיקה שלו וגם את חייו האישיים.

    תשובות לשאלות

    שלושת ההבדלים העיקריים בין Web 1.0, Web 2.0 ו-Web 3.0

    1. טבע התוכן והאינטראקציה:

    • Web 1.0 (רשת סטטית):
      • התוכן הוא סטטי וקריאה בלבד.
      • משתמשים יכולים רק לצרוך מידע ללא אינטראקציה.
      • דוגמאות: אתרי אינטרנט אישיים, דפי HTML סטטיים.

    • Web 2.0 (רשת חברתית):
      • התוכן הוא דינמי ואינטראקטיבי.
      • משתמשים יכולים ליצור, לשתף ולהגיב על תוכן.
      • דוגמאות: רשתות חברתיות (פייסבוק, יוטיוב), בלוגים, וויקי.

    • Web 3.0 (רשת סמנטית/מבוזרת):
      • התוכן מותאם אישית, אינטליגנטי ומבוזר.
      • למשתמשים שליטה רבה יותר על הנתונים ואינטראקציה עם יישומים מבוססי בינה מלאכותית.
      • דוגמאות: פלטפורמות מבוססות בלוקצ'יין, אפליקציות מבוזרות (DApps).

    2. בעלות ושליטה על נתונים:

    • Web 1.0:
      • הנתונים מאוחסנים בשרתים מרכזיים השייכים לאתרים פרטיים.
      • למשתמשים אין שליטה על הנתונים.
    • Web 2.0:
      • הנתונים עדיין מרוכזים, אך המשתמשים תורמים בצורה משמעותית (רשתות חברתיות, פלטפורמות תוכן).
      • הנתונים לרוב בבעלות ספקי הפלטפורמה.
    • Web 3.0:
      • הנתונים מבוזרים ונשלטים על ידי המשתמשים דרך בלוקצ'יין ורשתות עמית לעמית.
      • המשתמשים הם בעלי הנתונים ויכולים להחליט מי יקבל גישה.

    3. טכנולוגיה ומבנה:

    • Web 1.0:
      • מבוסס על HTML, דפים סטטיים וקישורים בסיסיים.
      • מוגבל באינטראקטיביות ואין תוכן שנוצר על ידי משתמשים.
    • Web 2.0:
      • משתמש ב-HTML5, CSS3, AJAX ומסגרות אינטראקטיביות (כמו React, Angular).
      • תומך ברשתות חברתיות, אינטראקציה עם משתמשים ו-APIs.
    • Web 3.0:
      • מבוסס על בלוקצ'יין, AI, למידת מכונה ונתונים סמנטיים.
      • מאפשר חוזים חכמים, אפליקציות מבוזרות וחוויות מותאמות אישית.

    תרחישי שימושי עבור כל Web

  • Web 1.0:
      • מציאת אתרים סטטיים המציגים תכניים כלליים ומתעדכנים לעיתים מאוד רחוקות, ללא אפשרות אינטרקציה עם העמוד.
  • Web 2.0:
      • המשתמש כותב או קורא תגובה של אנשים אחרים בעמוד.
  • Web 3.0:
      • מנוע החיפוש מעבד את השאלה בודק/סורק את האינטרנט ואת מסד הנתונים ומחזיר תשובה מותאמת למשתמש.
